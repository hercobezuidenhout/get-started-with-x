# Get started with X

The inspiration for this repo came from Codecrafter's [Build your own X](https://github.com/codecrafters-io/build-your-own-x).

In a nutshell, there are many different [Get Started](https://laravel.com/learn/getting-started-with-laravel) guides out there and *usually* at the end, you end up with some repository or small program.

After completing Laravel's [Getting started with Laravel](https://laravel.com/learn/getting-started-with-laravel) guide, I ended up with Chirper, a small Twitter clone. I have since looked at learning tech like Ruby, Jekyll, Rust, VS Code extensions, Google Chrome extensions, etc. and each guide will have me create some small project of sort.

Instead of living with thousands of Github repos, I decided to put all of these small get started guides into a single repository called **Get started with X**.
